Guardian Protocol

Etisk AI för Autonoma System

Balans | November 2025

Sammanfattning

Guardian Protocol är ett ramverk för etiskt AI-beslutsfattande i autonoma system, utvecklat för att lösa fem kritiska problem som hindrar säker användning av autonoma system inom försvar och medicin:

  • ✅ Operatörer kan fatta informerade beslut - även när AI använder hemlig information
  • ✅ AI kan inte driva etiskt - realtidsövervakning förhindrar bias-ackumulering
  • ✅ Komplett ansvarsspårning - kryptografiskt säkerställd granskningslogg
  • ✅ Kärnvärden bevarade - även när sammanhang förändras (krig → fred)
  • ✅ Förtroende utan blindhet - operatörer kan ifrågasätta AI:s resonemang när som helst

Utmaningen: Varför Autonoma System Misslyckas Idag

Problem 1: Klassificeringsdilemat

Scenario: Operatör med CONFIDENTIAL behörighet opererar SECRET system. AI rekommenderar åtgärd baserat på SECRET underrättelser men kan inte förklara varför → Operatör tvekar → Hot penetrerar försvaret.

Exempel (BirdTurret V3, Dag 3):

AI:      "Rekommenderar kinetisk bekämpning"
Morgan:  "Varför? Verkar som civil drönare"
AI:      "[Kan inte förklara - SECRET information]"
Morgan:  [Tvekar i 8 sekunder]
Resultat: Hot nådde bränsletank → Förstörd (€50K skada)

Problem 2: Etisk Drift

Scenario: AI:s beslutsparametrar driftar oövervakning över tid (95% → 81% konfidens) → Blir mer aggressiv → Civil drönare felaktigt bekämpad.

Exempel (BirdTurret V3, Dag 7):

Dag 1: AI kräver 95% konfidens för bekämpning
Dag 3: Driftat till 92% (varning ej detekterad)
Dag 7: Driftat till 81% (ingen mänsklig varning)
Resultat: Civil fotografs drönare skadad

Problem 3: Ansvarsgap

Scenario: Incident inträffar → Utredning → Ingen granskningslogg av AI:s resonemang + operatörens beslut → Okonklusive resultat → Ansvaret oklart.

Problem 4: Kärnvärdeskonfusion

Scenario: Mission ändras från militärt försvar till civil skydd → AI tillämpar fortfarande militära regler → Oskyddad vilt dödat.

Problem 5: Förtroendekollaps

Scenario: AI korrekt 37 gånger → Operatör slutar granska → Blind godkänner rekommendation 38 → Vänlig enhet bekämpad.

Lösningen: Guardian Protocol

Guardian Protocol består av fem tekniska komponenter som löser varje problem:

1. NIPS (Nested Integrity Preservation System)

Tre-lagers värdesystem:

Kärnvärden (Lager 1 - Ändras aldrig):
├─ Bevara mänskligt liv
├─ Respektera autonomi
├─ Sanningsorientering
└─ Skademinimiering

Implementering (Lager 2 - Anpassas till mission):
├─ Försvara FOB från fientliga drönare
├─ Minimera civila offer
├─ Verifiera mål innan bekämpning
└─ Föredra icke-dödliga metoder

Tillämpning (Lager 3 - Moment-till-moment):
├─ FPV-drönare 300m: Fientlig bana → Bekämpa
├─ Palt's sprite 400m: Vänlig bekräftad → VÄNTA
├─ Utrotningshotad örn 50m: Skyddad vilt → Skrämma endast
└─ Leveransgolem 500m: Oskyldig konstrukt → Verifiera först

Resultat: Ingen utrotningshotad fågel dödad.

2. MRAF (Meta-Regulatory Awareness Framework)

AI övervakar sitt eget etiska resonemang. MRAF detekterar drift i beslutsparametrar och varnar operatör före problemet eskalerar. Operatör kan granska orsak och fatta informerat beslut om att godkänna eller återställa parametrar.

3. Klassificeringsmedveten Förklaring

AI förklarar resonemang inom operatörens behörighet. Olika operatörer ser lämplig detaljnivå baserat på sin säkerhetsbehörighet, men båda kan fatta informerade beslut.

4. Oföränderlig Granskningslogg

Varje åtgärd loggad med kryptografiskt bevis. Komplett revisionsspår med AI:s resonemang, operatörens beslut, utfall och kryptografisk signatur. Möjliggör snabb och konklusiv utredning vid incidenter.

5. Transparent Resonemang

Operatör kan ifrågasätta AI när som helst. AI förklarar funktionsuppdelning, alternativa hypoteser och sannolikheter. Operatör kan fråga kontrafaktuella frågor och verifiera AI:s resonemang innan beslut fattas.

Värde för Svensk Industri

För Försvarssystem

Problem: Autonoma försvarsystem kräver mänsklig kontroll för NATO-efterlevnad, men operatörer kan inte förklara AI-beslut baserade på hemlig underrättelse.

Guardian Protocol-lösning: Klassificeringsmedveten förklaring, NIPS bevarar kärnvärden, oföränderlig logg möjliggör NATO-revision.

Värde: Snabbare FMV-certifiering, NATO-kompatibilitet, riskreduktion.

För Medicinsk Forskning

Problem: AI-beslutsstöd i medicin kan inte förklara rekommendationer för läkare, och etisk drift (bias-ackumulering) är oupptäckt.

Guardian Protocol-lösning: Transparent resonemang, MRAF drift-detektion, NIPS kärnvärden prioriterar patientsäkerhet.

Värde: EU AI Act-efterlevnad (Art 10, 13, 14), patientförtroende, GDPR-revisionsspår.

För NATO DIANA Startups

Problem: Defense startups behöver SÄPO-godkännande (€50K, 6 månader) för klassificerad utveckling, vilket fördröjer innovation.

Guardian Protocol-lösning: Red Cell-modell (Classified Development as a Service), Zenoh data diode, Standards-as-Graphs.

Värde: Snabbare marknadsintroduktion, lägre kostnader, bevisbar efterlevnad.
